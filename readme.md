# Project

## Installation

```sh

npm init -y


npm i colors                            // Colors
npm i inquirer                          // Inquirer
npm i dotenv                            // DotEnv


npm install sequelize mysql2            // ORM Sequelize
npm i jsonwebtoken                      // JSON webtoken

```

## Run Server

```sh

npm run dev
npm run db:connection                   // Test DB connection

```

## Opcional Handlebars

Fuente: https://github.com/pillarjs/hbs

```sh

npm install hbs

```
